#!/usr/bin/env python3
# ─────────────────────────────────────────────────────────────────────────────
# GeniusThinking — macOS Quick Action: Character Counter
#
# HOW TO INSTALL (takes 2 minutes):
#
# 1. Open Automator (Spotlight → "Automator")
# 2. File → New → choose "Quick Action"
# 3. At the top set:
#      "Workflow receives current"  →  "text"
#      "in"  →  "any application"
# 4. In the search bar on the left, search "Run Shell Script"
# 5. Drag "Run Shell Script" into the workflow area
# 6. Set Shell to: /bin/zsh
# 7. Set "Pass input" to: "as arguments"
# 8. Paste the script below into the box (replace the default content)
# 9. File → Save → name it "Count Characters"
#
# After saving:
# - Select any text anywhere on your Mac
# - Right-click → Services → "Count Characters"
# - A notification pops up with the count and target evaluation
#
# OPTIONAL — Add a keyboard shortcut:
#   System Settings → Keyboard → Keyboard Shortcuts → Services
#   Find "Count Characters" and assign any shortcut (e.g. ⌥⌘C)
# ─────────────────────────────────────────────────────────────────────────────

# ── PASTE THIS INTO THE AUTOMATOR "RUN SHELL SCRIPT" BOX ────────────────────
#
